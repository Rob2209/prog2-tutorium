# PROG2 Tutorium

Dieses Repository enthält alle Code-Beispiele und Übungsaufgaben aus dem PROG2-Tutorium.

## Struktur

Jedes Tutorium-Termin hat ein eigenes Package, nummeriert nach Reihenfolge:

```
t01_generics_wildcards/
├── examples/       # Fertige Beispiele aus dem Tutorium
├── exercises/      # Übungsaufgaben (mit TODO-Kommentaren)
└── solutions/      # Lösungen (werden nach dem Termin veröffentlicht)
```

## Verwendung

### Repository klonen

```bash
git clone https://github.com/Rob2209/prog2-tutorium.git
```

### Vor jedem Termin

```bash
git pull
```

Die neuesten Beispiele und Aufgaben sind dann lokal verfügbar.

## Hinweise

- **Lösungen** werden erst nach dem jeweiligen Termin in den `main`-Branch gemerged — vorher lohnt es sich, die Aufgaben selbst zu versuchen.
- Bei Fragen oder Fehlern gerne ein Issue öffnen.
